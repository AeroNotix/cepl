cepl
====

Code Evaluate Play Loop

My first lisp project, a chance to play about and get something up in opengl.

-----

**Requirements**

* cl-opengl
* lispbuilder-sdl
* varjo
* cl-utilities
* cl-ppcre
* symbol-munger

If you are using quicklisp then drop this in your local-projects directory and:
    (ql:quickload :cepl)
    (cepl:repl)
